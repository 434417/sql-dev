# Podsumowanie

W projekcie przedstawiono podstawy pracy z językiem SQL oraz systemem Microsoft SQL Server. Kurs zawiera teorię, przykłady zapytań i ćwiczenia praktyczne.

## Najważniejsze informacje

- SQL służy do komunikacji z relacyjnymi bazami danych.
- Dane w bazie są przechowywane w tabelach.
- `SELECT` pozwala pobierać i filtrować dane.
- `INSERT` dodaje nowe rekordy.
- `UPDATE` zmienia istniejące dane.
- `DELETE` usuwa rekordy z tabeli.
- `JOIN` łączy dane z wielu tabel.
- Funkcje SQL ułatwiają analizę danych, obliczenia i formatowanie wyników.

## Umiejętności po zakończeniu kursu

Po przejściu kursu użytkownik powinien umieć:

- rozpoznać podstawowe elementy relacyjnej bazy danych,
- napisać proste zapytanie `SELECT`,
- filtrować, sortować i grupować dane,
- dodawać, aktualizować i usuwać rekordy,
- łączyć tabele za pomocą `JOIN`,
- korzystać z podstawowych funkcji SQL Server,
- wykonać ćwiczenia na przykładowych danych.
