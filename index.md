# Kurs Microsoft SQL Server

Projekt realizowany w ramach przedmiotu Systemy Wspomagania Pracy Zespołowej.

## Cel projektu

Celem projektu jest przedstawienie podstaw języka SQL oraz sposobu pracy z systemem Microsoft SQL Server. Strona ma formę krótkiego kursu, który łączy teorię z przykładami gotowych zapytań.

## Zakres kursu

- podstawy relacyjnych baz danych,
- najważniejsze pojęcia języka SQL,
- obsługa Microsoft SQL Server,
- instrukcja `SELECT`,
- instrukcje `INSERT`, `UPDATE` oraz `DELETE`,
- łączenie danych z wielu tabel za pomocą `JOIN`,
- wybrane funkcje SQL,
- ćwiczenia praktyczne i przykładowe rozwiązania.

## Dlaczego SQL Server?

Microsoft SQL Server jest popularnym systemem zarządzania relacyjnymi bazami danych. Wykorzystuje się go w aplikacjach biznesowych, systemach internetowych, raportowaniu oraz analizie danych. Nauka podstaw SQL Server pozwala zrozumieć, jak przechowywać dane, pobierać je z tabel i modyfikować w kontrolowany sposób.
